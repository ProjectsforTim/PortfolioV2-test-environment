AI Autoencoder RF Anomaly Detection (Work in progress)

Built a deep learning-based anomaly detection model using an autoencoder trained on radio-frequency (RF) signal data. This project explores how neural compression and reconstruction error can identify unusual RF spectrum patterns, a technique useful for detecting potential drone or interference activity in noisy environments.
